“【python】win10下anaconda环境切换命令”

Published on: Apr 3, 2020
Tags: python

python|win10下anaconda环境切换命令

anacond下载地址:https://mirrors.tuna.tsinghua.edu.cn/anaconda/archive/

activate

activate 能将我们引入anaconda设定的虚拟环境中, 如果你后面什么参数都不加那么会进入anaconda自带的base环境,如果直接输入python则会直接进入base环境的python解释器，注意，此时命令行前面会多一个（base）说明此时我们处于了一个base环境之下。

1，创建一个新的虚拟环境

创建一个名为py36的虚拟环境，并且指定python的版本为3.6,conda会直接下载3.6的最新版本

conda create -n py36 python=3.6 或者 conda create –name py36 python=3.6

然后可以查看自己所建立的环境的位置

打开anaconda prompt,输入where conda可以看到本地的conda路径

2,切换版本

打开anaconda prompt

命令：activate name 例如activate py36

exit()退出python解释器

3，卸载环境

conda remove –name python36 –all

4,安装第三方包

输入 conda install gmpy2/conda install pycrypto

此时安装了gmpy2库和pycrypto库

5，卸载第三方包

命令：conda remove requests/ pip uninstall requests

6,查看环境包信息

conda list

7,导入导出环境

conda env export > environment.yaml

将包信息存入yaml文件中

当需要重新创建一个相同的虚拟环境时可以用：conda env create -f environment.yaml

以下是保存的一些命令，留作备用

activate // 切换到base环境

activate learn // 切换到learn环境

conda create -n learn python=3 // 创建一个名为learn的环境并指定python版本为3(的最新版本)

conconda list // 列出当前环境的所有包

conda install requests 安装requests包

conda remove requests 卸载requets包

conda remove -n learn –all // 删除learn环境及下属所有包

conda update requests 更新requests包

conda env export > environment.yaml // 导出当前环境的包信息

conda env create -f environment.yaml // 用配置文件创建新的虚拟环境
